HERMES CONTROL CENTER – BLUEPRINT FUER GOOGLE AI STUDIO
=======================================================

WAS IST DAS?
------------
Dieses Paket bereitet alles so weit vor, dass du in Google AI Studio daraus
gezielt ein fertiges Programm bauen lassen kannst.

IDEE DES PROGRAMMS
------------------
Eine zentrale Oberflaeche, die alles vereint:
- lokalen llama.cpp Server starten / stoppen / testen
- Hermes Endpoint verwalten
- GitHub anbinden
- Hugging Face anbinden
- Google AI Studio / Gemini API anbinden
- spaeter auch weitere OpenAI-kompatible Provider aufnehmen
- Profile, Logs, Status und Gesundheitspruefungen verwalten

WAS IST BEREITS VORBEREITET?
----------------------------
1. Projektkonzept
2. PRD / Produktanforderungen
3. Systemarchitektur
4. Ein Master-Prompt fuer Google AI Studio
5. API-Integrationsplan
6. Sicherheits- und Secret-Konzept
7. Beispiel-Konfigurationsdateien
8. OpenAPI-Backend-Blueprint
9. UI-Mockup als HTML
10. Code-Snippets fuer Adapter

REIHENFOLGE
-----------
1. Lies: 01_PROJEKTKONZEPT.md
2. Lies: 04_AI_STUDIO_MASTERPROMPT.md
3. Oeffne: ui_mockup.html
4. Nimm danach den Master-Prompt und verwende ihn in Google AI Studio
5. Nutze die restlichen Dateien als Referenz und Nachschlagewerk

WICHTIG
-------
Dieses Paket ist kein fertiges Endprogramm.
Es ist ein sehr weit vorbereiteter, strukturierter Bauplan,
mit dem du in Google AI Studio deutlich zielgerichteter zu einem
fertigen Programm kommst.
