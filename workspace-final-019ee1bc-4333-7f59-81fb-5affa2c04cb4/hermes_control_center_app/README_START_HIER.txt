HERMES CONTROL CENTER – START HIER
=================================

DAS IST DIE ERSTE ECHTE APP-STRUKTUR FUER DEIN SPAETERES PROGRAMM.

ENTHALTEN SIND:
- React Frontend
- Node.js Backend
- Electron Desktop-Huelle
- llama.cpp Serversteuerung
- Tray-/Autostart-/Benachrichtigungs-Grundlogik
- GitHub / Hugging Face / Gemini Integrationsgrundlagen
- Keynotes fuer Google AI Studio und andere KI-Systeme

WAS DU ZUERST LESEN SOLLTEST
----------------------------
1. README.md
2. docs/AI_KEYNOTES_PRODUKTANWENDUNG.md
3. docs/AI_SYSTEM_HINWEISE_FUER_WEITERENTWICKLUNG.md
4. docs/GOOGLE_AI_STUDIO_ODER_ANDERE_AI_PROMPT_ERGAENZUNG.md

WENN DU LOKAL ENTWICKELN WILLST
-------------------------------
1. npm install
2. START_DEV.bat

WENN DU EINE WINDOWS-APP BAUEN WILLST
-------------------------------------
1. npm install
2. BUILD_WINDOWS_APP.bat

WAS DIE APP TUN SOLL
--------------------
- lokalen llama.cpp Server starten / stoppen / ueberwachen
- Hermes-Anbindung vereinfachen
- GitHub, Hugging Face und Gemini integrieren
- Profile, Logs und Konfigurationen zentral verwalten

WICHTIGER HINWEIS
-----------------
Dies ist ein GRUNDGERUEST, kein komplett fertiges Produkt.
Es ist aber so vorbereitet, dass du es
- lokal weiterentwickeln,
- von Google AI Studio erweitern lassen,
- oder durch andere KI-Systeme zielgerichtet ausbauen lassen kannst.
